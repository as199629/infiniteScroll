# 簡介

這是一個使用 Vue 3、Vite 和 Vuetify 構建的專案。
專案內容為實作無限滾動，並介接 github repositories api。

# 建置專案

在開始使用之前，請確保已經安裝了 Node.js 和 npm。

## 安裝依賴

npm install

## 在開發模式下啟動專案

npm run dev
